# SkyDelves

A World of Warcraft addon that tracks and displays bountiful delves for Midnight (patch 12.0).

## Features

- Shows only bountiful delves currently available
- Compact, sleek UI with minimize/maximize
- Lock/unlock frame movement
- Auto-updates when completing delves
- Remembers position and visibility between sessions
- Only displays for max-level characters

## Installation

1. Download or clone this repository
2. Copy the `SkyDelves` folder to `World of Warcraft\_retail_\Interface\AddOns\`
3. Restart WoW or reload UI

## Usage

- `/sd` or `/skydelves` - Toggle window
- Click `-` to minimize, `+` to maximize
- Click `L`/`U` to lock/unlock frame movement
- Click `×` to close window

## Requirements

- World of Warcraft: Midnight (12.0.1 or higher)
- Max level character
